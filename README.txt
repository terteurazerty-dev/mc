Casino Keep Villagers (Fabric server mod)

What it does
- Tags Casino Rocket villagers with scoreboard tag: casino_keep
- Rewrites commands that target minecraft:villager so they exclude tag=!casino_keep
  (keeps Academy's villager removal active for normal villagers)

Build (Java 21)
- ./gradlew build
Output jar: build/libs/casino-keep-villagers-<version>.jar

Install
- Put the built jar into your SERVER mods/ folder and restart.
